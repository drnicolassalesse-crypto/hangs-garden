# Task 3-5 — PWA Polish & Install UX

## Goal
Close out the PWA story: Android Chrome install prompt integration, iOS Safari "Add to Home Screen" instructions, splash/status-bar styling, and an SW update toast.

## Scope
- `src/features/install/InstallPrompt.tsx` — listens for `beforeinstallprompt`, stashes the event, and renders a small bottom-bar "Install Planta" CTA. On tap: call `.prompt()`, handle `userChoice`, and hide permanently (localStorage `planta.install.promptShown`) after acceptance or dismissal.
- `src/features/install/IOSInstallHint.tsx` — feature-detects iOS Safari (non-standalone). Shows a one-time hint at the bottom of Today with the Add-to-Home-Screen steps; dismissible via localStorage `planta.install.iosHintDismissed`.
- `src/features/install/detect.ts` — `isStandalone()`, `isIOSSafari()`, `canInstallPromptSupported()`.
- `src/App.tsx` — mount both components; they self-decide visibility.
- `src/registerSW.ts` — use `virtual:pwa-register` `onNeedRefresh` / `onOfflineReady` callbacks; show a tiny toast when a new version is available with an "Update" button that calls `updateSW(true)`.
- `src/features/install/UpdateToast.tsx` — renders the toast above the bottom bar.
- `index.html` — add apple-mobile-web-app-status-bar-style `default`, `theme-color` already set, add a `<link rel="mask-icon">` fallback and viewport `viewport-fit=cover` (already present).
- `public/manifest.webmanifest` — already generated by vite-plugin-pwa; no edits.
- README update: short "Install" section explaining both paths.

## Acceptance criteria
- On a Chromium browser that fires `beforeinstallprompt`, tapping "Install Planta" shows the native prompt and hides the bar after the user's choice.
- On iOS Safari (non-standalone), an on-screen hint appears with the Share-icon → Add-to-Home-Screen steps, and is dismissible once.
- When a new SW is available, a toast appears with an Update button; tapping it refreshes the page on the new version.
- `npm run build` produces a `sw.js`, `manifest.webmanifest`, and precaches the app shell; `npm run typecheck` clean.

## Out of scope
- Full splash image generation for iOS (`apple-touch-startup-image`) — optional polish
- Play Store / TWA packaging
